◆ setChildAlign()

void ui::Column::setChildAlign ( align_t  align)

Set where to start when displaying child widgets; on the top, or bottom.

If align is ui::ALIGN_TOP, then child widgets are rendered from top to bottom. If align is ui::ALIGN_BOTTOM, then child widgets are rendered from bottom to top.

Note
ui::ALIGN_CENTER is undefined behavior and should not be used, for the sake of clarity at least.
